LifeLines

  •   
    This lifeline is known as Flip/Switch the Question. If a player chooses to mark/use this lifeline, he/she will be presented with a new question. Having said that, player still needs to answer the current question, the outcome of which (right or wrong) will not impact the game.
  •   
    This lifeline is known as Ask the Audience. If a player chooses to mark/use this lifeline, he/she will be presented with a system generated projection (graph) of the opinion of imaginary audience.
  •    This lifeline needs no explanation. If you are not aware of this lifeline, chances are that you might be living under a rock. Get a life!
  •    This lifeline is known as Double chance. If a player chooses to mark/use this lifeline, he/she will have the opportunity to make two guesses in case the first guess is wrong.
  •    This lifeline is known as Extra Time.If a player chooses to mark/use this lifeline, the timer will be incremented by 30 extra seconds.
  •    This lifeline is known as Ask the Expert. If a player chooses to mark/use this lifeline, the system will do a google search of the question. The timer will not stop on marking this lifeline.

Timing & others!

  • Each Question will be timed. A running timer of 45 seconds is shown while the player plays the game. If a player fails to register an answer within the allotted time, the game will be reset automatically.
  • There are three Padaav (safe zones) in the game, which occurs at 80,000 , 12,50,000 and 1 Crore (10 million). Clearing these safe zones guarantees the prize money (imaginary and not real).
  • All the lifelines are single use. Once a lifeline has been marked, can not be undone. Therefore, it is advisable to be very judicious while using a lifeline. You probably are aware of this but I needed to make it official.
  • The questions currently come from a configuration file. However, there are plans to integrate a backend or get the question from some other method.
  • If you would like to feed your own questions and play with your friends, please navigate to Upload Questions screen. The Upload Questions screen has option for either using the system user interface to upload questions or you can upload the json file itself.
  • For any other emergency either rush to a hospital or contact cops.
